IDEA常用插件

发表于 2020-03-31 分类于 工具使用 阅读次数： 本文字数： 241 阅读时长 ≈ 1 分钟

IDEA 常用插件

Lombok

提供 @Data，@builder，@ToString，@NoArgsConstructor 等相关注解，可以很大程序上减少代码维护的负担

Save Actions

可以自动保存代码，并且在保存的时候做一些额外的操作

Material Theme UI

超级多的主题样式

Alibaba Cloud Toolkit

高效开发并部署适合在云端运行的应用

Free MyBatis plugin MyBatisX

mapper 到 mapper.xml 相互跳转

Rainbow Brackets

彩虹括号

Grep Console

控制台颜色显示

CodeGlance CodeGlance Pro

嵌入一个代码小地图

Maven Helper

依赖分析及排除冲突

HighlightBracketPair

高亮显示成对的括号

Alibaba Java Coding Guidelines

代码规范检查，使用 Alibaba Java 开发规约

IDE Eval Reset

重置 IDEA 试用时间

Settings -> Plugins -> Manage Plugin Repositories 添加 https://plugins.zhile.io

CheckStyle-IDEA

代码规范检查，可以使用自定义代码风格文件
https://github.com/checkstyle/checkstyle/blob/master/src/main/resources/google_checks.xml

RestfulTool

搜索 Restful 请求，快捷键 Ctrl + Alt + /
